# Car recognition app

[Server] Python (Flask)

[IOS, Android] ReactNative, TypeScript

## Project structure
```
app/ (mobile app)
server/ (api for neural network)
```

## Getting Started

### Install
```
brew update
brew install node
npm install -g expo-cli
```

## Build and Run

```
git clone https://github.com/AndreiSukharev/vtb_hack_mobile.git hack
```

### Server
```
cd hack/server
npm i
npm run dev
```
### Mobile application
```
cd hack/app
npm i
expo start
follow the instruction in the browser to launch mobile app in expo environment
```
